# Tiny_JSON
## 概述
+ 用于参考和研究。
+ 参考教程[1][2]，使用C++11（原教程使用C89）实现一个符合标准的JSON解析器和生成器。
+ 保证Windows平台的顺利编译使用，后续会实现跨平台。  

## 语言&环境
+ C++ 11 (STL)
+ Windows C++ (Visual Studio)
+ gtest 

## 特性
+ 仅支持UTF-8 
+ 仅支持double的number类型
